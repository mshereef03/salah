تحميل النسخة الإلكترونية (PDF)

🔧 تعليمات سريعة:

  1. ارفع هذا الملف باسم index.html إلى مستودعك في GitHub (مثلاً salat-book).
  2. ضع ملف الـ PDF بجانب هذه الصفحة باسم book.pdf أو غيّر الرابط أعلاه ليتطابق مع اسم الملف.
  3. من إعدادات المستودع → Pages → اختر Deploy from branch، ثم فرع main والمجلد / (root) واحفظ.
  4. خلال نصف دقيقة تقريباً ستحصل على رابط مثل: https://yourname.github.io/salat-book/

تلميح للاستدامة طويلة المدى: اجعل رمز الاستجابة السريعة (QR) يشير إلى رابطك الدائم على نطاقك الخاص مثل https://salah.guide/pdf ثم وجّه هذا الرابط إلى ملف GitHub Pages/Release.
